2)what is the default style of text in pretag?

The default style of text within a pre tag in HTML is rendered in a monospaced (or fixed-width) font. This means that each character, including spaces, takes up the same amount of horizontal space. Additionally, the text is displayed exactly as it is written in the HTML, preserving all whitespace, line breaks, and indentation.

3)how text position has been given in a table in html task?

In HTML, the position of text within a table cell can be controlled using CSS properties like text-align and vertical-align. Here's how they work: text-align: Controls the horizontal alignment of text within a table cell. left: Aligns text to the left. center: Centers the text. right: Aligns text to the right. vertical-align: Controls the vertical alignment of text within a table cell. top: Aligns text to the top. middle: Centers the text vertically. bottom: Aligns text to the bottom.
